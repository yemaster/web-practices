# web-practices

CTF web 新手从零练习题库的源代码（包括题目附件和 Dockerfile）。

[**在线训练**](https://ctf.u5tc.cn)

一般来说，附件题中的 `archive.zip` 是题目下载的附件，然后其余文件是题目的源代码。容器题目的 `xxxx.tar` 是容器打包后的文件，其余文件是题目的源代码。

部分题目因为题目描述不清等原因在之后可能会更新，另外之后可能会补充题目的题解。
